1. Create `src/components/ui/BottomSheet.tsx` by evolving Modal.tsx:
   - Keep existing props: `isOpen`, `onClose`, `title`, `children`
   - Add optional `detent` prop: `'peek' | 'half' | 'full'` (default `'half'`)
   - Replace `slideUp` variant animation with `useMotionValue`-driven y position:
     - `y = useMotionValue(window.innerHeight)` — starts off-screen
     - Detent pixel offsets: peek = `vh * 0.6`, half = `vh * 0.5`, full = `vh * 0.1`
     - Guard for jsdom: `const vh = window.innerHeight || 600`
     - On open (`isOpen` becomes true): `animate(y, DETENTS[detent], { type: 'spring', ...gentle })`
     - On close: `animate(y, vh, { type: 'spring', ...snappy })` then call `onClose`
   - Add `useDrag` from `@use-gesture/react` on the handle bar div ONLY (not the content):
     - `axis: 'y'`, `filterTaps: true`
     - During active drag: `y.set(Math.max(DETENTS.full, currentDetentY + my))`
     - On release: dismiss if `my > vh * 0.3` OR velocity `vy > 0.5` (500px/s in px/ms)
     - Otherwise snap back: `animate(y, currentDetentY, { type: 'spring', ...snappy })`
   - Drag-to-dismiss only when content scrolled to top: use `contentRef.current?.scrollTop === 0` check before allowing drag
   - Backdrop opacity: `useTransform(y, [DETENTS.full, vh], [0.5, 0])` — import from `motion/react` NOT `motion/react-m`
   - Keep: body scroll lock, Escape key, auto-focus, ARIA attributes
   - Apply `{...bind()}` to the handle bar container div (the `<div className="flex justify-center pt-3 pb-1">` element), add `touch-action: none` style
   - The `m.div` sheet container uses `style={{ y }}` instead of `variants={slideUp}`
   - Keep `AnimatePresence` wrapping for mount/unmount (backdrop fade)

2. Delete `src/components/ui/Modal.tsx` (replaced by BottomSheet.tsx).

3. Update all 4 consumers to import BottomSheet instead of Modal:
   - `TaskModal.tsx`: `import BottomSheet from '../ui/BottomSheet'` — replace `<Modal` with `<BottomSheet detent="half"` (task form fits in half)
   - `EatingWindowConfig.tsx`: same import update — `detent="half"`
   - `MealTemplateEditor.tsx`: same import update — `detent="full"` (complex form with food items)
   - `ProtocolEditor.tsx`: same import update — `detent="full"` (supplements list + timing rules)
   - In each file: only the import line and component name change. Props are identical.

Critical constraints:
- Import `useMotionValue`, `useTransform`, `animate` from `'motion/react'` — NOT from `'motion/react-m'`
- Import `m` from `'motion/react-m'` for animated DOM elements
- `bind()` goes on the handle bar div; `style={{ y }}` goes on the sheet `m.div`

- `overflow: hidden` on the sheet wrapper to prevent content flash during mount animation
